# Icons
Icons is the use of images and symbols to portray a subject, movement or ideal.
In general, the most basic function of the icons is to convey the information quickly.
People fix and perceive pictorial elements like icons and illustrations faster than words.

## Getting Started
You can start using it by importing the icon, like:
```tsx
// Import the icon individually
import IcChecklist from '@/components/icons/IcChecklist';

// Import multiple icon on one import
import { IcEyeOpen, IcEyesClose } from '@/components/icons';
```

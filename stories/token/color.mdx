# Color ðŸŽ¨
Color token are used to set the color of every aspect of your application, either it text, background, border color, etc.
These tokenization system is based on material color system where 500 is the primary color.

## Getting Started
You can start using it by implementing it in className in format \{type syntax} - \{color name} - \{token}.
For example if you want to have a background with primary color 500, the className would be "bg-primary-500".

```tsx
// usage example
<div className="bg-n-1 p-5 rounded-xl border border-solid border-n-5">
  ...
</div>
```
